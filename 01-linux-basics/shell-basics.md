## Core Shell Commands

| Command        | Description               | How it works                          | Pentester Tip |
|---------------|---------------------------|---------------------------------------|---------------|
| `Ctrl+Alt+T`  | Open terminal             | Launches default terminal emulator    | Always work from terminal |
| `clear`       | Clear screen              | Clears terminal buffer                | Avoid confusion |
| `history`     | Show command history      | Reads shell history file              | Look for leaked creds |
| `history -c`  | Clear history             | Removes command traces                | OPSEC after engagement |
| `!123`        | Run command by number     | Executes history entry                | Speed up tasks |
