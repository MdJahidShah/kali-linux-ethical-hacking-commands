Kali Linux Ethical Hacking Commands

A structured and practical reference of Kali Linux commands and tools used in ethical hacking and penetration testing.

This repository is designed to help cybersecurity learners, bug bounty hunters, and penetration testers understand what each command does, how it works internally, and where it fits in real-world hacking methodologies.

The repository covers information gathering (OSINT), network scanning, enumeration, web application testing, password auditing, wireless security, exploitation, and post-exploitation analysis using Kali Linux.

All examples focus strictly on authorized security testing, professional workflows, and ethical learning practices.
