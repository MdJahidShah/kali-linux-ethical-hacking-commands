Linux File Editing Commands

Practical file editing in Kali Linux for ethical hackers and penetration testers.

1. Nano Editor

Nano is a beginner-friendly command-line text editor.

  • nano filename.txt - Open or create a file.
  • Use Ctrl + O to save and Ctrl + X to exit.
  • Tip: Always check permissions before editing critical files.

2. Vim Editor

Vim is a powerful editor for advanced users.

  • vim filename.txt - Open a file.
  • Switch to insert mode: i, save & exit: Esc :wq
  • Tip: Learn basic navigation: h, j, k, l.

3. Other Useful File Commands

  • cat file.txt - Display file content.
  • less file.txt - Scroll through a file page by page.
  • touch newfile.txt - Create an empty file.
  • cp source.txt dest.txt - Copy files.
  • mv oldname.txt newname.txt - Move or rename files.
  • rm file.txt - Remove files (be careful!).

4. Pentester Tips

  • Always make backups of important configuration files before editing.
  • Use ls -la to check hidden files before modifying anything.
  • Check file permissions with stat file.txt before writing changes.
